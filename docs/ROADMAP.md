# Roadmap

The roadmap is directional. GitHub issues and milestones should become the source of truth once the repository is created.

## v0.1 — Foundation

- Career dashboard and marathon archive
- Individual race pages
- Standardized pre-race training windows
- Weekly mileage visualization
- Race comparison page
- Versioned Garmin and Strava-derived datasets

## v0.2 — Trusted analytics

- Move all cycle calculations into tested analytics modules
- Add import validation and data-quality reporting
- Add rolling mileage and long-run progression
- Introduce Fitness, Durability, Specificity, and Freshness definitions
- Show calculation details and confidence notes in the UI

## v0.3 — Performance genome

- Test athlete-specific training hypotheses
- Identify recurring successful and unsuccessful patterns
- Model taper response and medium-long-run density
- Add course, weather, injury, and non-goal-race context
- Maintain evidence-backed athlete “laws” with supporting races and confidence

## v0.4 — Live training cycle

- Import new Garmin or Strava exports incrementally
- Detect the current goal race and training phase
- Compare current training with historical builds
- Surface data gaps, fatigue signals, and actionable weekly observations

## v1.0 — Prediction and coaching

- Produce an explainable finish-time range
- Quantify uncertainty and limiting factors
- Recommend adjustments grounded in historical response
- Add race execution and pacing analysis
- Preserve human coaching judgment instead of presenting model output as certainty
